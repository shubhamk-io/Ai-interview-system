# 🚀 AI Interview SaaS Platform

> **Powered by OpenAI API for real-time AI-driven interview generation and intelligent feedback.**

🔗 **Live Repository:** https://github.com/shubhamk-io/Ai-interview-system

A production-ready, full-stack SaaS application that simulates real-world technical and HR interviews using artificial intelligence.

This is not a basic CRUD project — it is a complete **AI-powered platform** with authentication, payments, and scalable architecture.

---

## 🌟 Overview

The **AI Interview SaaS Platform** enables users to:

* Upload resumes (PDF)
* Generate AI-based interview questions
* Practice real interview scenarios
* Receive intelligent feedback

The platform leverages the **OpenAI API** to dynamically generate questions and analyze user responses, creating a highly personalized interview experience.

---

## ✨ Key Features

* 📄 **Resume Upload (PDF Support)**
  Upload resumes for personalized interview generation

* 🤖 **AI-Generated Questions (OpenAI API)**
  Context-aware interview questions powered by AI

* 🎯 **Technical & HR Interview Practice**
  Simulates real-world interview rounds

* 🧠 **AI Feedback & Analysis**
  Smart evaluation of answers with improvement suggestions

* 💳 **Credit-Based SaaS System**
  Controlled feature access using credits

* 💰 **Razorpay Payment Integration**
  Secure credit purchase system

* 🔐 **Firebase Google Authentication**
  Fast and secure login

* 🎨 **Modern UI/UX**
  Smooth animations using Framer Motion

* ☁️ **Production Deployment**
  Fully deployed on Render

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash id="clone001"
git clone https://github.com/shubhamk-io/Ai-interview-system.git
cd Ai-interview-system
```

### 2️⃣ Backend Setup

```bash id="backend001"
cd server
npm install
npm run dev
```

### 3️⃣ Frontend Setup

```bash id="frontend001"
cd client
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file in the server folder:

```env id="env001"
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key

# OpenAI
OPENAI_API_KEY=your_openai_api_key

# Firebase
FIREBASE_API_KEY=your_key

# Razorpay
RAZORPAY_KEY_ID=your_key
RAZORPAY_SECRET=your_secret
```
